Student Portal System

Description: The Student Portal System is a backend-driven academic management tool designed using Python and SQL. It provides a structured environment to store, retrieve, and manage essential academic data like student profiles, enrolled courses, and subject details. The system mimics real-world institutional databases, giving hands-on experience with relational data modeling and CRUD operations.

Core Features:

  • 👤 Student Management: Register, update, and view student profiles
  • 📚 Course Enrollment: Track which courses each student is enrolled in
  • 📖 Subject Database: Add, update, and retrieve subject details under each course
  • 🔍 Search Functionality: Lookup students by ID, name, or enrolled subject
  • 💾 Data Storage: Fully integrated with SQL for persistent data storage
  • 📊 Structured Queries: Perform joins between student, course, and subject tables for advanced querying
